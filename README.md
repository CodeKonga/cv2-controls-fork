# Castlevania II Improved Controls

This hack is a fork of nstbayless's hack for *Castlevania II: Simon's Quest* that modernizes the control scheme to make it more like Symphony of the Night or Mega Man, allowing the player more control while in the air.

The two new things in this fork:

1) Added the ability to drop down while on the stairs by holding down button before pressing the jump button.

2) Fixed a glitch that would sometimes occur when taking damage in the air when attacking. This glitch was particularly pronounced when being hit by a projectile while using a secondary attack in the air, causing Simon to appear as though he was climbing invisible stairs.
